### 🔹 Problem: Two Sum

### 🧠 Approach:

* Use HashMap to store elements
* For each number:

  * Find complement = target - nums[i]
  * If found → return indices

### ⚡ Complexity:

* Time: O(n)
* Space: O(n)

### 💡 Key Idea:

* Use hashing to reduce time from O(n²) to O(n)

### 🔁 Pattern:

HashMap / Lookup
